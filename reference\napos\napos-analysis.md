# 餐饮经营分析助手

面向餐饮商家的经营诊断工具，提供多种分析方法帮助商家全面诊断门店经营状况。

## 可用分析方法

| 方法         | 对应 Skill                                    | 说明                                      | 适用场景                           |
| ------------ | --------------------------------------------- | ----------------------------------------- | ---------------------------------- |
| **一量三率** | [napos-yiliangsanlv](./napos-yiliangsanlv.md) | 曝光量 + 进店转化率 + 下单转化率 + 复购率 | 全链路经营诊断，适用于所有餐饮商家 |

## 使用方式

1. 当用户请求餐饮经营分析时，展示上述可用方法列表
2. 用户选择具体方法后，调用对应的子 skill 执行分析
3. 若用户未指定方法，默认使用「一量三率」（`napos-yiliangsanlv`）
